# Brian Thomas Jr. — media drop

Put the photos / video clips you find for this video **in this folder**
(`public/brian-thomas/`). It's gitignored, so nothing here gets committed.

## Naming (so I can map each asset to a beat)

- **Video clips** → `clip1.mp4`, `clip2.mp4`, … (a highlight, a catch, etc.)
- **Photos** → `photo1.jpg`, `photo2.jpg`, …
- **Voiceover** → `vo.m4a` or `vo.wav` (record from the script; I'll convert m4a→wav)

## Format tips

- **Vertical (9:16) is ideal**, but any aspect works — I'll crop/position under the charts.
- Video: `.mp4` or `.mov`. Photos: `.jpg` / `.png`.
- Don't worry about editing them — just drop the raw files and tell me what each one is
  (e.g. "clip1 = the deep TD catch"). I'll composite the charts + captions on top.

Once files are here, tell me and I'll wire them into the composition and render.
